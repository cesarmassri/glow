# Glow: Neon Arcade Matcher 👾

¡Bienvenido a **Glow**! Un juego arcade minimalista, rápido y brillante, creado con pura energía y *VibeCoding*.

## 🎮 Jugar ahora
Puedes probar la versión en vivo del juego aquí:
> [https://cesarmassri.github.io/glow](https://cesarmassri.github.io/glow)

## 🚀 Sobre el juego
Glow es un juego de reflejos y ritmo. El objetivo es simple: dispara a las bolitas de colores que caen por el canal central antes de que lleguen al fondo.

* **Coincidencia de colores:** Si disparas el color correcto, la bolita explota. Si fallas, la bolita acelera y se complica el juego.
* **Dificultad progresiva:** Cada 100 puntos, la velocidad aumenta. ¡Mantente alerta!
* **Audio Arcade:** Sonidos sintéticos generados en tiempo real mediante la Web Audio API.

## ⌨️ Controles
El juego es totalmente responsivo y funciona tanto en PC como en dispositivos móviles.

| Acción | Teclado | Botón en Pantalla |
| :--- | :--- | :--- |
| **Disparar Rojo** | `1` o `R` | Botón **R** |
| **Disparar Verde** | `2` o `G` | Botón **G** |
| **Disparar Azul** | `3` o `B` | Botón **B** |

## 🛠️ Stack Tecnológico
Este proyecto fue construido desde cero con:
* **JavaScript (Vanilla):** Lógica de juego y manejo de estados.
* **HTML5 Canvas:** Renderizado de gráficos con estética neón.
* **Web Audio API:** Generación de sonidos (sin archivos externos).
